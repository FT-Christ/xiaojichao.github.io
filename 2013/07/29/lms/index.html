尝试采用Node技术开发LMS系统

LMS系统本来打算采用Java开发，基于Restful Web Services架构，后台采用Spring MVC、Hibernate/myBatis，前端采用ArgularJS，UI采用FlatUI for BootStrap。

现在决定改用Node.JS开发。

  1. Node.js+Express实现Restful Web Services

  2. MongoDB存储数据，Mongoose实现持久层

  3. 前端采用Angular.js

  4. UI采用FlatUI for BootStrap

  5. 模块加载采用RequireJS

  6. 采用Nginx做反向代理
