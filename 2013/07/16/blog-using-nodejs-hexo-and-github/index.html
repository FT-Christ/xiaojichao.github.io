使用node.js、hexo、github搭建博客

今天参考zipperary的博客，用node.js和hexo，在github上搭建了自己的博客。整个过程比用github pages要顺利很多，以后就在这里写前端相关的博客了。

关于hexo，可以参考hexo的官方网站，上面解释的很清楚。

在Windows平台上写博客，建议用装个MarkdownPad。在hexo下，运行hexo new建立博客后，到/source/_posts目录下，用MarkDownPad打开编辑。编辑完存盘，在generate、deploy。
